# Book Haven 

Live Website: https://book-haven-57fb6.web.app/

## Website Features

- Browse and discover books from different genres
- User login system with Google and email options  
- Add new books 
- Manage your personal book collection
- Responsive design for mobile and desktop,Laptop

## Technology Used

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js  
- Database: MongoDB
- Hosting: Firebase

## How to Use

1. Visit the website
2. Create account or login
3. Browse books
